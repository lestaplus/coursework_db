# Внески команди

## Учасник 1: Реуцький Артем

- Реалізація функціоналу, пов’язаного з користувачами, підписками та платежами
- Реалізація сценарію реєстрації користувача з підпискою з використанням транзакцій
- Реалізація оновлення статусу підписки з перевіркою терміну дії
- Реалізація мʼякого видалення користувачів із збереженням історії платежів
- Реалізація простих SELECT-запитів для користувачів та підписок
- Реалізація складного аналітичного запиту з аналізу доходів по типах підписок за місяцями
- Написання інтеграційних тестів для функціоналу користувачів, підписок та платежів
- Написання unit-тестів для сервісів користувачів, підписок та платежів

## Учасник 2: Шевцов Анатолій

- Налаштування проєкту та початкової інфраструктури
- Проєктування та реалізація функціоналу для книг, авторів, жанрів та позичок
- Реалізація сценарію додавання книги з авторами та жанрами з використанням транзакцій
- Реалізація оновлення статусу позички з бізнес-валідацією
- Реалізація жорсткого видалення прострочених позичок
- Реалізація простих SELECT-запитів для книг і позичок
- Реалізація складного аналітичного запиту для визначення найпопулярніших книг
- Написання інтеграційних тестів для функціоналу книг та позичок
